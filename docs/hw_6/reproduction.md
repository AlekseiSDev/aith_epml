# Инструкции по воспроизведению

Для полного воспроизведения результатов проекта выполните следующие шаги:

## 1. Подготовка окружения

Убедитесь, что у вас установлен [uv](https://github.com/astral-sh/uv).

```bash
# Клонирование репозитория
git clone https://github.com/AlekseiSDev/aith_epml.git
cd aith_epml

# Создание окружения и установка зависимостей
uv sync --extra dev
```

## 2. Получение данных и моделей

Проект использует DVC для версионирования данных.

```bash
# Инициализация DVC (если нужно) и получение данных
# Примечание: требуется доступ к Google Cloud Storage remote
dvc pull
```

## 3. Запуск пайплайна

Для запуска полного цикла (сплит данных -> обучение -> тест) используется Luigi:

```bash
uv run python -m wine_quality_epml.pipeline.runner --local-scheduler
```

## 4. Отслеживание экспериментов

Эксперименты автоматически логируются в ClearML. Убедитесь, что у вас настроены учетные данные в `~/clearml.conf` или через переменные окружения.

## 5. Сборка документации

```bash
# Локальный просмотр
uv run mkdocs serve

# Сборка статического сайта
uv run mkdocs build
```
